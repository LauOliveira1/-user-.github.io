Atenção: Consulte a agência antes de realizar qualquer alteração nesses campos

Através dos campos abaixo é possóvel realizar a alteração dos blocos da página principal, podendo selecionar qualquer banner extra (do 1 ao 11) para exibição, construir uma lista de produto personalizada utilizando filtros e muito mais. faça uma cópia e salve o Conteúdo abaixo antes de realizar qualquer alteração!.

Preencha seguindo as orientações abaixo:
  • 1. Posição: "topo", "meio" ou "baixo"
  • 2. Tipo do banner: extra1 ~ extra12
  • 3. Ativar Slider?: "slider" ou em branco (O cadastro do banner deve conter mais de uma imagem)
  • 4. Classe CSS Colunas: "col" + 1~12 + "Em Branco", "sm", "md", "lg"
  • 5. Classe CSS personalizada: "classe_css" ou em branco

Tabelas de Medidas & Provador
Atenção!
Ao preencher as Tabelasm de Medidas a mesma será utilizada na criação da Tabela de Medidas e tambem no Provador Virtual como base do calculo.
Siga os padrões corretamente para que as funcionalidades Não parem de funcionar e sempre mantenha um Backup da ultima versóo.

INFORMAR O NOME DA MARCA EXATAMENTE COMO NO CADASTRO
Tabelas de medidas por MARCA em formato JSON
padrões
  1. Sempre finalizar um BLOCO com virgual (,). Um BLOCO é o conteudo entre as chaves ({ e })
  2. Sempre usar aspas (") nos NOMES DAS MARCAS
  3. Sempre usar aspas (") nos TAMANHOS
  4. Sempre usar aspas (") nas LETRAS DAS MEDIDAS
  5. SIGNIFICADOS DAS LITRAS
  6. b = Medida máxima do busto
  7. c = Medida máxima da cintura
  8. q = Medida máxima do quadril
  9. As medidas devem ser informados no formato de NúmeroS INTEIROS e sem as aspas (")
